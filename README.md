# Password list generator

## Very fast

## Changes in code

### Located at top of script

### Prefix

Prefix is a string before the numbers

Example:

```powershell
prefix00001

prefix00002

...

prefix99999
```

### Number count

#### Usually recommended for all digits to be 9

Defines how many numbers are there after prefix

example:

```javascript
const numberCount = 999
```

Output:

```powershell
prefix001

...

prefix999
```
